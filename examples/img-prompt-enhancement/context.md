# Luminth Landing Page Hero — Image Prompt Context

## What is Luminth?

Luminth is a modular, AI-assisted world-building and interactive storytelling
platform. Tagline: "Most tools help you write a world. Luminth helps you run one."

## Target Audience

Game Masters, DMs, writers, world-builders, and indie game developers who run
tabletop RPGs and collaborative fiction.

## Three Modes

- **Build** — Author worlds with AI-assisted generation. Entity editors, canon
  tools, timeline management.
- **Test** — Validate world coherence before players find the gaps. Consistency
  checks, scenario simulations.
- **Play** — In-character interaction with NPCs that remember. Multi-NPC scenes,
  session tools, story state tracking.

## Visual Identity

- **Palette**: Dark warm (brown-black backgrounds, teal/gold accents, parchment text)
- **Fonts**: Cormorant Garamond serif headings + DM Sans body
- **Aesthetic**: "Fantasy-meets-technology" — like an illuminated manuscript built
  by engineers
- **Inspiration**: D&D Beyond (illustrated fantasy vibrancy) + Raycast/Stripe
  (polished modules, bento grids)

## Genre Versatility

The hero cycles through 7 genre themes: Fantasy, Sci-fi, Cyberpunk, Western,
Horror, Pirate, and custom worlds. Each with unique fonts, colors, imagery,
and evocative world hooks. The landing page must communicate that Luminth
handles ALL genres, not just fantasy.

## Hero Image Requirements

- Full-width banner for luminth.ai landing page
- Must leave space for text overlay (title + tagline)
- Dark background that transitions/fades at edges
- Should feel vibrant and alive, not flat or corporate
- Must suggest multiple genre worlds coexisting
- Should evoke wonder and creative possibility
